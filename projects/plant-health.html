Plant Health Monitoring

This project detects plant diseases using a Convolutional Neural Network (CNN) and provides solutions for treatment. The backend, built using Django REST Framework, was deployed on AWS for scalability.

  • Django REST Framework
  • Convolutional Neural Networks (CNN)
